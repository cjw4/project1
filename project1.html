Project 1. This is the first project for my AJAX class at MATC. The goal is to get familiar with dynamic HTML by implementing it in form validation. The form validation works as follows.

  • All changes happen when the validate button is clicked.
  • Any invalid field will result in an inline error message (See below).
    • I added some Regular Expression validation.
    • Name entered must be at least 2 letters long with no spaces.
    • Phone number and zip code must be numbers and correct length.
  • If the invalid field is filled in validly the error message will disappear.
  • If the form is completely valid the inputed values will show up below the form.
